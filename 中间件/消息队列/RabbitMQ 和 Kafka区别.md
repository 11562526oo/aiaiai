| 对比   | RabbitMQ         | Kafka             |
| ---- | ---------------- | ----------------- |
| 架构   | Exchange + Queue | Topic + Partition |
| 吞吐量  | 万级/s             | 百万级/s             |
| 消息存储 | 消费后删除            | 日志持久化             |
| 顺序性  | 队列顺序             | Partition顺序       |
| 使用场景 | 业务系统             | 大数据流处理            |
