# Linux 基础命令速查

## 1. 目录与文件操作

```bash
# 当前路径
pwd

# 列出文件（含权限、大小）
ls -al

# 切换目录
cd /path/to/dir

# 创建目录（带多级）
mkdir -p /opt/app/logs

# 拷贝文件/目录
cp a.txt b.txt
cp -r src/ dest/

# 移动/重命名
mv old.txt new.txt

# 删除文件/目录（慎用）
rm file.txt
rm -rf /path/to/dir
```

## 2. 权限相关

```bash
# 查看权限
ls -l

# 修改权限
chmod 755 run.sh
chmod +x run.sh

# 修改所有者
chown user:group file.txt
```

## 3. 进程与端口

```bash
# 查看 Java 进程
ps -ef | grep java

# 根据端口查进程
lsof -i:8080
netstat -tlnp | grep 8080
```
## 4.查找日志命令
```bash
#查看 某一行日志的上下相关行数
grep -n '企业的域名迁移工单已执行完毕' app.log     -- 12345
```
