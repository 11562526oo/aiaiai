	特征工程是将原始数据转换为可以更好地表示问题的特征形式，帮助模型更好地理解和学习数据中的规律
1. 特征选择（特征选择不会创建新特征，也不会改变数据结构）
	1. 过滤法
	2. 包裹法
	3. 嵌入法
2. 特征转换（对数据进行数学或统计处理，使其变得更加适合模型的输入要求）
	1. 归一化
	2. 标准化
	3. 对数变化
	4. 类别变量的编码
		1. 独热编码
3. 特征构造
	(特征构造是基于现有的特征创造出新的、更有代表性的特征。通过组合、转换、或者聚合现有的特征，形成能够更好反映数据规律的特征。)
4. 特征降维
	1. 主成分分析（PCA）
	2. 线性判别分析（LDA）

**常用方法**
1. 低方差过滤法 
	低于阈值的特征的数据直接清除掉   sklearn
	```
import numpy as np  
  
a = np.random.randn(100)  
print(np.var(a))  
print(a)  
#%%  
b = np.random.randn(100) * 0.1  
print(np.var(b))  
print(b)  
#%%  
c = np.random.normal(5,0.1,size =100)  
# 构建数据样本  
X = np.vstack((a,b)).T  
print(X.shape)  
#%%  
from sklearn.feature_selection import VarianceThreshold  
# 低方差过滤  
vt = VarianceThreshold(0.01)  # 创建一个过滤器  
X_new = vt.fit_transform(X)  
print(X_new.shape)
	```
2. 相关系数法
	（1）皮尔逊相关系数
			正相关   负相关   无关
	```
import numpy as np  
import pandas as pd  
  
advertising = pd.read_csv("../data/advertising.csv")  
print(advertising)  
print(advertising.shape)  
advertising.drop(advertising.columns[0], axis=1, inplace=True)  
print(advertising)  
print(advertising.shape)  
  
#%%  
# 去掉控制  
advertising.dropna(inplace=True)  
print(advertising.isnull().sum())  
print(advertising)  
#%%  
X = advertising.drop("Sales",axis=1)  
print(X.shape)  
Y = advertising["Sales"]  
print(Y.shape)  
print(X.corrwith(Y, method="pearson"))  
  
print(advertising.corr(method="pearson"))  
#%%  
import seaborn as sns  
import matplotlib.pyplot as plt  
# 计算皮尔逊相关系数矩阵  
corr_matrix = advertising.corr(method="pearson")  
  
# 可视化热力图  
sns.heatmap(corr_matrix, annot=True, cmap="coolwarm", fmt=".2f")  
plt.title("Feature Correlation Matrix")  
plt.show()
	```
