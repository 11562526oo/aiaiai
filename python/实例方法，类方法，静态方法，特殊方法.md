# 一、实例方法（Instance Method）

### 1. 定义与特点

- 在类中**普通定义**的方法（没有 `@classmethod`、`@staticmethod` 装饰器）。
- 第一个参数**约定**为 [self]，代表**当前实例对象本身**。
- 常用调用方式：[实例对象.方法名(...)]
- 也可以用类调用：[类名.方法名(实例对象, ...)]（不推荐，影响可读性）。

### 2. 能做什么？

在实例方法里面，可以：

- 访问 / 修改 **实例属性**：[self.name]、[self.age]
- 访问 **类属性**：[self.**class**.attr] 或 [类名.attr]
- 调用 **类方法 / 静态方法 / 其他实例方法**

### 3. 示例

复制

```markdown
class Person:
    # 类属性
    kind = "human"

    def __init__(self, name, age):
        # 实例属性
        self.name = name
        self.age = age

    # 实例方法
    def introduce(self):
        print(f"我叫 {self.name}，今年 {self.age} 岁，是 {self.__class__.kind}")
        # 等价于 Person.kind
```

调用：

复制

```python
p = Person("小王", 18)
p.introduce()        # 推荐写法
Person.introduce(p)  # 等价写法，但不推荐
```

# 二、类方法（Class Method）

### 1. 定义与特点

- 使用装饰器 [@classmethod] 定义。
- 第一个参数**约定**为 [cls]，代表**类本身**（不是实例）。
- 可以通过：
    - [类名.类方法(...)]（推荐）
    - [实例.类方法(...)]（语法上可以）

### 2. 能做什么？

在类方法里面，可以：

- 访问 / 修改 **类属性**：[cls.attr]
- 用作**工厂方法**：根据不同参数返回该类的实例 [return cls(...)]
- 实现与“整个类”相关的逻辑，而不是某一个对象的逻辑。

### 3. 示例

复制

```markdown
class Person:
    kind = "human"
    count = 0

    def __init__(self, name, age):
        self.name = name
        self.age = age
        Person.count += 1

    @classmethod
    def show_count(cls):
        print(f"当前创建了 {cls.count} 个 Person 实例")

    @classmethod
    def from_string(cls, info: str):
        # 工厂方法：通过字符串创建对象
        name, age = info.split(',')
        return cls(name, int(age))
```

调用：

复制

```python
p1 = Person("小王", 18)
p2 = Person.from_string("小李,20")

Person.show_count()  # 推荐
p1.show_count()      # 也可以，但底层 cls 仍然是 Person
```

# 三、静态方法（Static Method）

### 1. 定义与特点

- 使用装饰器 [@staticmethod] 定义。
- **没有** [self]，也**没有** [cls] 参数。
- 更像是“定义在类内部的普通函数”，只是为了**归类管理**、语义清晰。
- 可以通过：
    - [类名.静态方法(...)]
    - [实例.静态方法(...)]

### 2. 能做什么？

- 不依赖具体实例，也不依赖类本身，只是和这个类“有逻辑上的关系”的一段功能代码。
- 常用于：
    - 工具函数（格式校验、简单计算等）
    - 与类相关但不需要访问类 / 实例数据的逻辑

### 3. 示例

复制

```markdown
class Person:
    @staticmethod
    def is_adult(age):
        return age >= 18
```

调用：

复制

```python
print(Person.is_adult(20))  # True
p = Person()
print(p.is_adult(16))       # False，语法上也可以，但不推荐这样用
```

# 四、特殊方法 / 魔术方法（Special / Magic Methods）

所谓“特殊方法”，就是名字两边带有**双下划线**的那些方法，例如 [**init**]、[**str**] 等。  
它们**不会**直接由你调用，而是**在特定场景下由 Python 解释器自动触发**。

常见模式：  
[**方法名**] → 有特定的“钩子”作用。

---

## 1. [**init**]：构造初始化

- 作用：创建实例时自动调用，用于对实例属性进行初始化。
- 触发时机：[对象 = 类名(...参数...)]

复制

```markdown
class Person:
    def __init__(self, name, age):
        print("正在初始化 Person 对象...")
        self.name = name
        self.age = age

p = Person("小王", 18)  # 会自动调用 __init__
```

---

## 2. [**str**]：人类可读的字符串表示

- 作用：返回一个“给人看的”字符串，当你 [print(obj)] 时会用它。
- 触发时机：[print(obj)] 或 [str(obj)]

复制

```markdown
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def __str__(self):
        return f"Person(name={self.name}, age={self.age})"

p = Person("小王", 18)
print(p)          # 自动调用 __str__，输出：Person(name=小王, age=18)
```

---

## 3. [**repr**]：官方/调试用的字符串表示

- 作用：返回一个“尽量精准、可用于调试”的表示。
- 通常期望：[eval(repr(obj))] 能重新创建这个对象（理想情况）。
- 触发时机：在交互式解释器里直接敲变量名、或者 [repr(obj)]

如果没有定义 [**str**]，[__repr__] 会在 [print] 时被用作替代。

---

## 4. 运算相关的特殊方法

### [**add**]：加法运算 [+]

复制

```markdown
class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __add__(self, other):
        return Vector(self.x + other.x, self.y + other.y)

    def __str__(self):
        return f"Vector({self.x}, {self.y})"

v1 = Vector(1, 2)
v2 = Vector(3, 4)
v3 = v1 + v2      # 自动调用 v1.__add__(v2)
print(v3)         # Vector(4, 6)
```

### 常见运算相关方法（了解）：

- [**sub**]：减法 [-]
- [**mul**]：乘法 [*]
- [**truediv**]：除法 [/]
- [**eq**]：相等比较 [==]
- [**lt**]：小于 [<]，等等

## 5. 容器与下标相关方法

### [**len**]：内置函数 [len(obj)]

复制

```markdown
class MyList:
    def __init__(self, data):
        self.data = data

    def __len__(self):
        return len(self.data)

ml = MyList([1, 2, 3, 4])
print(len(ml))    # 自动调用 ml.__len__()，结果 4
```

### [**getitem**]：支持下标访问 [obj[key]]

复制

```markdown
class MyList:
    def __init__(self, data):
        self.data = data

    def __getitem__(self, index):
        return self.data[index]

ml = MyList([10, 20, 30])
print(ml[1])  # 自动调用 ml.__getitem__(1)，输出 20
```

还有：

- [**setitem**]：赋值 [obj[key] = value]
- [**contains**]：["x in obj"]

## 6. 上下文管理：[__enter__] 与 [__exit__]

用于 [with] 语句。

复制

```markdown
class MyResource:
    def __enter__(self):
        print("资源打开")
        return self

    def __exit__(self, exc_type, exc_val, exc_tb):
        print("资源关闭")

with MyResource() as r:
    print("使用资源中...")
```

输出顺序：

复制

```text
资源打开
使用资源中...
资源关闭
```

# 五、三种方法 + 特殊方法 总对比表

|类型|装饰器|第一个参数|典型调用方式|主要操作对象|典型用途|
|---|---|---|---|---|---|
|实例方法|无|[self]|[obj.method()]|某个具体**实例**|读写实例属性，定义对象的行为|
|类方法|[@classmethod]|[cls]|[Class.method()]|整个**类**|操作类属性、工厂方法、类级逻辑|
|静态方法|[@staticmethod]|无|[Class.method()]|都不是（纯函数）|工具函数、与类相关但无需类/实例数据|
|特殊方法|系统约定（双下划线）|通常是 [self]|由 Python 自动触发|依据具体方法而定|控制对象的创建、显示、运算、容器行为等|
